# 🍓 BerryLens — Определение ягод с Gemini AI

Минималистичное Android/iOS приложение для определения ягод и фруктов по фотографии, построенное на Expo + Gemini Vision API + Material You.

---

## 📱 Экраны

- **Главный** — камера с видоискателем, кнопки «Снять» и «Галерея»
- **Результат** — название, съедобность, описание, вкус, польза, предупреждения
- **История** — последние 20 сканирований
- **Настройки** — ввод Gemini API ключа

---

## 🚀 Сборка APK (шаги)

### 1. Установи Node.js и Expo CLI

```bash
# Node.js: https://nodejs.org (рекомендуется v18+)
npm install -g expo-cli eas-cli
```

### 2. Войди в Expo аккаунт (бесплатно)

```bash
# Создай аккаунт: https://expo.dev/signup
eas login
```

### 3. Установи зависимости

```bash
cd BerryLens
npm install
```

### 4. Инициализируй EAS проект

```bash
eas init
# При запросе — согласись создать новый проект
```

### 5. Собери APK

```bash
# APK (можно установить напрямую, без Google Play)
eas build --platform android --profile preview
```

Через ~10-15 минут получишь ссылку на скачивание APK. Сборка происходит в облаке Expo — ничего дополнительно устанавливать не нужно!

## 🔑 Получить Gemini API ключ (бесплатно)

1. Открой [aistudio.google.com](https://aistudio.google.com/app/apikey)
2. Войди через Google аккаунт
3. Нажми **Get API key → Create API key**
4. Скопируй ключ
5. В приложении → Настройки → вставь ключ

**Лимиты бесплатного плана:** 1500 запросов/день, 15 запросов/минуту

## ⚙️ Технологии

- **Expo SDK 51** + **Expo Router** — навигация на основе файловой системы
- **expo-camera** — доступ к камере
- **expo-image-picker** — выбор из галереи
- **Gemini 1.5 Flash** — быстрый и бесплатный Vision API от Google
- **Material You** — цветовая система Google (тёмная тема)
- **AsyncStorage** — хранение ключа и истории локально

## ❓ FAQ

**Ошибка «Неверный API ключ»** — убедись что ключ начинается с `AIzaSy` и введён без пробелов

**«Ягоды не обнаружены»** — сфотографируй ближе, убедись что ягоды/фрукты хорошо видны

**Камера не открывается** — разреши доступ к камере в настройках Android
